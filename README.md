# Claude Code Skills Repository

This repository contains reusable skills for [Claude Code](https://claude.ai/claude-code).

## Available Skills

### solidjs-patterns

Best practices and patterns for SolidJS development, including reactivity, state management, performance optimization, and testing.

## Installation

Install a skill using the `skills` CLI:

```
npx skills add https://github.com/omniaura/skills --skill solidjs-patterns
```
